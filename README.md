# **BlinkIt Dynamic Sales Dashboard – Power BI**

Created by **Hellen-caren Dereck**

This project is an interactive Power BI dashboard designed to analyze BlinkIt’s sales performance across different outlet types, product categories, and customer segments. The dashboard makes it easy to explore sales trends, ratings, item distribution, and outlet characteristics using a clean and fully interactive interface.

---

## **Dashboard Highlights**

* **Total Sales:** $1.20M
* **Average Sales:** $141
* **Total Items Sold:** 8,523
* **Average Rating:** 3.9

---

## **Key Features**

* KPI cards for a quick performance overview
* Sales trend analysis by outlet establishment year
* Item type comparison using bar charts
* Fat content distribution (Low Fat vs Regular)
* Outlet size and outlet location insights
* Outlet type breakdown with total sales, item counts, ratings, and visibility scores
* Fully interactive filter panel:

  * Outlet Location Type
  * Outlet Size
  * Item Type

---

## **What This Dashboard Shows**

* Best-performing outlet types and sizes
* Top-selling product categories
* Differences in sales by outlet location tier
* Customer ratings and item visibility patterns
* Overall trends across years and categories
